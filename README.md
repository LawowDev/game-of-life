# Compiling
make sure you have ncurses installed on your machine.
if it's not, check here to install it : [Linux Terminal](https://fr.linux-terminal.com/?p=5124) (in French)

to compile, execute :
'''sh
cc main.c -o game-of-life -lncurses
'''
then just run
'''sh
./game-of-life
'''
